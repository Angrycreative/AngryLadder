# Angryladder

Based on Lumen

```
composer install
```


## API endpoints

```
/v1/games/ (get)
/v1/games/{id} (get)
/v1/games/ (post)
/v1/games/{id} (put)

/v1/players/ (get)
/v1/players/{id} (get)
/v1/players/top (get)
/v1/players/top/mostgames (get)
/v1/players/ (post)
/v1/players/{id} (put)
```
